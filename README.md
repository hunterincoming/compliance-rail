# Compliance Rail

Private, day-one provenance layer for cannabis genetics built on Midnight.

## Goal
Give breeders and operators a way to cryptographically timestamp and prove ownership of new genetics on day one — without ever exposing the raw data.

## Current Focus
Building the private timestamp + provenance layer. Longer-term vision includes support for enforceable licensing terms and royalty tracking.

## Status
Early stage. Currently validating with operators and labs.
